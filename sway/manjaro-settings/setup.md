# Setup Manjaro

## Setup Links
```bash
# Config
ln -s ~/git/dotfiles/sway/manjaro-settings/config config

# Defaults (confis from Manjaro-Sway build
ln -rs ~/git/dotfiles/sway/manjaro-settings/defaults .

# User's config
ln -rs ~/git/dotfiles/sway/config.d .

# User's definitions
ln -rs ~/git/dotfiles/sway/definitions.d .

```
